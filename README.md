# Trader Behavior & Sentiment Analysis

## Project Overview
This project analyzes trader behavior by combining trading performance metrics with market sentiment indicators. The goal is to understand how sentiment influences profitability and decision-making.

## Key Analysis Performed
- Total and Daily Profit & Loss (PnL)
- Buy vs Sell performance comparison
- Win rate by trading side
- Cumulative PnL analysis
- Relationship between Fear & Greed Index and PnL

## Tools & Technologies
- Power BI
- Python (Pandas, Matplotlib)
- Excel / CSV datasets

## Project Files
- `Sentimental_trading_analysis_dashboard.pbix` – Power BI dashboard  
- `analysis.ipynb` – Data cleaning & analysis notebook  

## Key Insights
- Buy and Sell sides show different risk-return characteristics
- Extreme fear or greed levels significantly impact trader profitability
- Consistent strategies outperform high-risk trades over time

## How to Use
1. Download the `.pbix` file and open in Power BI Desktop
2. Open `analysis.ipynb` to review data processing and analysis steps
